YaPiG as every software has this programming mistakes called bugs, unfortunatelly YaPiG, has thousands :(.

I would appreciate help with the English text included on Yapig's documentation and on Yapig's Home Page this is because my mother language is Spanish and I have my English a little oxidized. So, please, if you find any incorrection on these texts, let me know it! Thank you!

The best way to report a sofware bug is to fill the form of YaPig's Bug Tracking system. Please, before posting try to see if any other user already submited the same bug. On the Bugtrack, bugs have three categories, they can be recognised by the background's title color: darkest red means that those bugs are being solved or have been solved only on the CVS version. Middle red bugs have a pending solution and lightest red background means that those bugs are solved on the unstable version.

You can also send me an email to natasab@users.sourceforge.net, but I would prefer that you use the Bug Track System
